Detailed Description

The PyDeviceInterface class represents a interface module in the GenTL module hierarchy.

This class represents an individual physical interface in the System. For example, a network interface card (NIC) for GigE Vision connection, CXP or Camera Link frame grabber board, or USB3 Vision driver in the GenTL system. The PyDeviceInterface class includes methods to enumerate the available devices on the physical interface in the system.

IpxCamPy.PyDeviceInterface Python class is wrapper around C++ class IpxCam::Interface

This method re-enumerates the devices.

The ReEnumerateDevices method allows the user to re-enumerate the devices connected to the Interface and update the list of PyDeviceInfo objects returned by subsequent GetDeviceInfoList() method calls.
