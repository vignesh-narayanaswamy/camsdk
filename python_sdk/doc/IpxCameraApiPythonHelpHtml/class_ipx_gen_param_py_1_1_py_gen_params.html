Detailed Description

An PyGenParams class contains methods to access all GenICam camera parameters.

This class contains methods that can access each node from the GenICam camera description XML file by parameters type and name.

Py::Object IpxGenParamPy::PyGenParams::GetRootCategory ( )

This method gets the object of the root category node. The Root node is considered a special node. It has no parent node. In the topology graph, it is the top node which connects to at least one child node. The child node may connect to the device node that provides the connection to the transport layer.

Returns
If the method succeeds, it returns the IpxGenParamPy.PyGenParamCategory object of the root category node. Returns None if failed
