Cheetah Imperx Camera SDK Samples


KAC Cheetah Imperx Camera Samples

The samples illustrate on how to use the Imperx Camera SDK. These samples are based on Qt 5.3 and written in C++.

Cheetah Imperx Camera SDK Samples

Description
QtCheetahKACExposureUI Demonstrates on how to use the Imperx Camera SDK functions to initialize and call the GrabbingThread, IpxCam(System, Device,Stream, DeviceInfo, IpxDisplay) modules. The user will become familiar with the methods to set the Exposure Mode, Exposure in microseconds(us), enable the fixed frame period, setting the frame time in microseconds, and setting the frame rate in fps.
QtCheetahKACGainUI Demonstrates on how to use the Imperx Camera SDK functions to initialize and call the GrabbingThread, IpxCam(System, Device,Stream, DeviceInfo, IpxDisplay) modules. The user will become familiar with the methods to set the Analog Gain, Digital Gain, and Black Level.

NOTE:You must be login as root to modify the usbfs_memory_mb if you need to display an image greater than 7MB. You can modify the size of the usbfs memory mb if needed by editing the 'usbfs_memory_mb' file. For example:

sudo sh -c "echo 1000 > /sys/module/usbcore/parameters/usbfs_memory_mb"
