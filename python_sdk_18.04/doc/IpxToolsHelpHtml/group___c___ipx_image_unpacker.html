This C-interface function unpacks the input raw source IpxRAWImage to the targeted output destination IpxImage.

Parameters
[in]hImageUnpackerPointer of the IpxHandle for the IpxUnpacker
[in]sourcePointer to the raw source IpxRawImage
[out]outputPointer to the output destination IpxImage
Returns
Returns the error code:
  • IPX_ERR_OK Successfully unpacks the source IpxImage to the targeted output destination IpxImage.
  • If IpxError error code < 0, then it returns a negative error code indicating problems unpacking the IpxImage
